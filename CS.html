C# Language

C# (pronounced C-Sharp) is a modern, object-oriented programming language developed by Microsoft. It is mainly used for building Windows applications, web applications, and games. C# is known for its strong typing, good performance, and ease of use, especially for developers working with the .NET framework.

        if(age >= 18)
        {

            Console.WriteLine("Minor");

        // Loop - for

History

C# was developed by Microsoft in the late 1990s as part of the .NET initiative. The language was led by Anders Hejlsberg, a well-known software engineer.

C# was officially released in 2000 with the first version of the .NET Framework. It was designed to be a modern, object-oriented, and type-safe language, combining the power of C++ with the simplicity of Visual Basic.

Since then, C# has evolved significantly with new versions introducing features like LINQ, async/await, generics, and more, making it popular for desktop applications, web development, and game development (especially with Unity).

Documentation

C# documentation is officially maintained by Microsoft and is part of the .NET platform. It provides detailed explanations of the C# language, its syntax, and how it works within the .NET framework.

The documentation includes a language reference that explains keywords, data types, operators, and control structures. This section is especially useful for beginners who want to understand how C# code is written and structured.

In addition, the documentation contains a full API reference that lists all available classes, methods, and namespaces. Developers can use it to learn how built-in features like Console.WriteLine() and other .NET libraries work.

Microsoft also provides tutorials and guides that help developers build real applications such as console programs, web applications, and games. These guides include examples that can be tested directly in an IDE like Visual Studio or JetBrains Rider.
